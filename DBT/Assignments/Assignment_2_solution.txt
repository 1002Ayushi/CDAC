4) Why isn’t it possible to see the first five rows of a table?
If you're unable to see the first five rows of a table in a database, there could be several reasons:

No Data in the Table: The table might be empty, meaning there are no rows to display.

Permission Issues: You may not have the necessary permissions to view the data in the table.

Incorrect Query: The SQL query you are using might not be properly structured to retrieve the first five rows.
